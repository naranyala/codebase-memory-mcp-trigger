# codebase-memory-mcp-trigger

A Pi extension that provides a `/codebase-memory-mcp` command to trigger the `codebase-memory-mcp` binary.

## Installation

```bash
pi install git:https://github.com/your-username/codebase-memory-mcp-trigger
```

## Usage

Run the command in your Pi session:

`/codebase-memory-mcp <args>`

If no arguments are provided, it defaults to `--help`.
